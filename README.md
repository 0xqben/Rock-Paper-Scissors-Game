# Rock Paper Scissors — C++

Final project for **Course 5: Algorithms and Problem Solving — Level 2**, part of the *Foundations of Programming* roadmap by Dr. Mohammed Abu-Hadhoud (Programming Advices).

## About the Project

This is a console-based Rock Paper Scissors game built in C++.

## My Process

1. **Read the requirements only** — I went through Dr. Abu-Hadhoud's project requirements without watching his solution or looking at any reference code.
2. **Built it independently** — I implemented the full game using the concepts covered throughout the course, aiming for clean, well-structured code.
3. **Compared and refined** — After finishing my own version, I watched Dr. Abu-Hadhoud's walkthrough of the project, identified gaps in my solution, and refactored my code to align with the more professional approach he demonstrated.

## Concepts Used

- Enums
- Structs
- Function decomposition
- Forward declarations
- Input validation

## What I Learned

Working through this project reinforced how to break a problem into small, testable functions and how to structure a program using enums and structs instead of relying on loose variables. Comparing my independent solution against the instructor's version was especially useful for learning cleaner, more professional patterns.
